Navigation au sein d'un fichier en pseudo-code

Pour pouvoir naviguer à travers les lignes de pseudo-code, AlgoWaker permet d'effectuer plusieurs types de mouvements. Ceux-ci sont identiques peu importe le mode choisi, mais ils s'effectuent de manière sensiblement différente ; ils se font dans les deux cas au clavier. Ces mouvements sont : Avancer, Reculer, Aller à la ligne n et Quitter.

Mode CUI - Console

En mode console, il suffit d'entrer la lettre qui correspond au mouvement choisi quand le programme demande d'entrer une action. Les combinaisons sont respectivement :

  • - [Entrée] = Avancer : Permet de passer à la ligne suivante dans le programme
  • - [B] + [Entrée] = Reculer : Permet de remonter d'une ligne dans le programme
  • - [Ln] + [Entrée] = Aller à la ligne n : Permet d'atteindre la ligne n dans le programme

Mode GUI - Graphique

En mode graphique, les actions possibles restent les mêmes. Toutefois, elles ne se tapent plus dans une zone de saisie utilisateur mais se déclenchent par la combinaison de certaines touches. Les voici :

  • - [Entrée] = Avancer
  • - [CTRL] + [B] = Reculer
