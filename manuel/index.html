Accueil

Qu'est-ce que le "Guide du galérien" ?

C'est est un guide.

À quoi sert-il ?

Il permet de t'éviter des heures de galère en te présentant les fonctionnalités du fantastique programme AlgoWaker.
Les catégories du menu horizontal ci-dessus t'amèneront vers des explications pour chacune des opérations possibles avec le programme.
Dans chaque cas, les opérations te seront présentées selon deux versions : le mode CUI Console User Interface : C'est un mode qui s'exécute dans une fenêtre en lignes de texte utilisable au clavier uniquement. et le mode GUI Graphical User Interface : C'est un mode plus classique, il s'exécute dans une fenêtre normale utilisable à la souris et au clavier..

À propos du programme AlgoWaker

Ce programme a été créé par l'équipe Win'Rs en janvier 2017.
AlgoWaker est compatible avec les systèmes Windows, Linux et MacOS et requiert l'installation de Java.
