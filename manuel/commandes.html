Commandes

Pour pouvoir naviguer à travers les lignes de pseudo-code et effectuer d'autres actions, AlgoWaker permet d'entrer des commandes à (presque) chaque instant de l'exécution. Les mouvements sont : Avancer, Reculer et Aller à la ligne n. Les autres actions sont Ouvrir le manuel utilisateur et Quitter.

Il suffit d'entrer la lettre qui correspond à l'action choisie quand le programme demande d'entrer une action. Les possibilités sont :

  • - [Entrée] = Avancer : Permet de passer à la ligne suivante dans le programme
  • - [B] + [Entrée] = Reculer : Permet de remonter d'une ligne dans le programme
  • - [Ln] + [Entrée] = Aller à la ligne n : Permet d'atteindre la ligne n dans le programme
  • - [O] + [Entrée] = Ouvrir un autre fichier que celui passé en paramètre
  • - [H] + [Entrée] = Ouvrir le manuel utilisateur dans le navigateur internet
  • - [Q] + [Entrée] = Quitter le programme

Si l'action entrée n'est pas reconnue, le programme l'ignore simplement et attend une autre saisie.
